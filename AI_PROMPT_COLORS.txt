TalkTimes 프로젝트 색상 규칙 - AI 프롬프트용

=== 기본 원칙 ===
- 미니멀 디자인 우선, 과도한 색상 사용 금지
- WCAG 2.1 AA 기준 대비율 4.5:1 이상 준수
- 정의된 색상 팔레트만 사용, 임의 색상 추가 금지
- 라이트/다크 모드 동시 지원

=== 색상 팔레트 ===

[배경 색상]
- 메인 배경: #F9FAFB (라이트) / #1C1C1E (다크)
- 섹션 배경: #F5F5F5 (라이트) / #2A2A2A (다크)

[텍스트 색상]
- 메인 텍스트: #121212 (라이트) / #F0F0F0 (다크)
- 서브 텍스트: #4B5563 (라이트) / #9CA3AF (다크)
- 보조 텍스트: #9CA3AF (라이트) / #4B5563 (다크)

[카드/컨테이너 배경]
- 기본 카드: #FFFFFF (라이트) / #181818 (다크)
- 강조 카드: #EFEFEF (라이트) / #2C2C2E (다크)

[테두리/구분선]
- 기본 테두리: #E5E7EB (라이트) / #3A3A3C (다크)
- 강조 테두리: #D1D5DB (라이트) / #4A4A4C (다크)

[포인트/액션 색상]
- Primary (메인 액션): #3B82F6
- Success (성공/완료): #10B981
- Warning (경고/주의): #F59E0B
- Error (오류/삭제): #EF4444
- Neutral (비활성/보조): #CBD5E1

=== TalkTimes 특화 색상 ===
- 뉴스 정치: #3B82F6
- 뉴스 경제: #10B981
- 뉴스 IT/기술: #F59E0B
- 뉴스 스포츠: #EF4444
- 뉴스 문화: #CBD5E1
- 브랜드 컬러: #3B82F6

=== 사용 규칙 ===
DO:
- 위 정의된 색상만 사용
- 텍스트 대비율 4.5:1 이상 유지
- 의미에 맞는 색상 사용 (성공=녹색, 오류=빨간색)

DON'T:
- 순수 검정(#000000) 텍스트 사용 금지
- 한 화면에 포인트 색상 3개 이상 사용 금지
- 색상만으로 정보 전달 금지

=== AI 코딩 시 적용 방법 ===
1. Tailwind CSS 사용 시 색상 코드를 직접 지정
2. CSS Variables나 상수로 색상 정의하여 재사용
3. 다크모드는 조건부 클래스나 CSS 변수로 처리
4. 컴포넌트 props로 색상 테마 전달 시 정의된 색상만 허용

예시:
- 메인 버튼: bg-[#3B82F6] text-white
- 카드 배경: bg-[#FFFFFF] dark:bg-[#181818]
- 메인 텍스트: text-[#121212] dark:text-[#F0F0F0]
